# Secure Coding Review Tool

A simple web-based tool that analyzes source code for common security vulnerabilities.

## Features

- Detects hardcoded passwords
- Identifies weak passwords
- Detects SQL Injection risks
- Detects unsafe eval() usage
- Generates a security score
- Provides security recommendations

## Technologies Used

- HTML
- CSS
- JavaScript

## How to Run

1. Download the project files.
2. Open `index.html` in a web browser.
3. Paste code into the text area.
4. Click **Analyze Code** to view results.
